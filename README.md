# LLM Architecture Evolution

An offline-friendly, mobile-responsive study guide covering the evolution of
large language model architectures from statistical language models through
Transformers, mixture-of-experts, retrieval, multimodality, and test-time
compute.

## Study features

- 23 chapters organized into six learning paths
- Architecture timeline and concept map
- Accessible SVG diagrams and comparison tables
- Searchable glossary
- 46 flashcards and 30 quiz questions
- Browser-local progress tracking
- Responsive, printable, dependency-free HTML
